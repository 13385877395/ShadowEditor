Shadow Editor

简介

  • 名称：Shadow Editor
  • 版本：v0.0.2
  • 说明：Three.js自带编辑器的二次开发。

主要功能

  1. 3D场景在线编辑。
  2. 各种类型3D数据的导入导出。
  3. 材质编辑。
  4. 支持js脚本。
  5. 各种光源和雾效。
