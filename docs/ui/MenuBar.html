菜单栏

菜单栏

由场景、编辑、二维、几何体、光源、资源、组件、播放、工具、选项、帮助几个菜单组成。

场景

新建

新建场景，当前场景会被清空。可以创建空场景和GIS场景。

保存

保存当前场景。如果是新场景，保存时会提示输入场景名称。如果是载入场景，则直接保存到原场景上。

另存为

将当前场景另存为一个新场景。

发布场景

将当前场景发布为静态文件，包含场景用到的所有页面、脚本、资源。方便部署在IIS、Apache、Tomcat等服务器软件上。

编辑

撤销

撤销上一步操作。

重做

重做撤销的操作。

清空历史记录

清空所有撤销、重做操作。清空侧边栏历史面板中的所有历史记录。

复制

将当前选中的物体复制为一个新物体。

删除

删除当前选中的物体。

二维（开发中）

为场景提供二维控件。例如：按钮、标签、面板、水平线、条形图、时间、竖直线、日期、时间圆盘、键值标签、表单、仪表、柱状图、折线图、侧边栏、散点图、饼状图、弦图、力导向图、树状图、集群图、包图、分区图等。
