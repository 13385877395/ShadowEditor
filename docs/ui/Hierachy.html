场景层次图

场景层次图

场景层次图用树来表示场景中各物体的相对关系。

选中物体

单击树上的节点选中物体，并在属性面板显示物体属性。

切换视角

双击树上的节点切换到该物体视角。

设置层次关系（有bug）

将一个物体拖动到另一个物体上，可以设置后一个物体是前一个物体的子元素。

删除物体

选中树上的节点，按Del键，可以删除物体。
