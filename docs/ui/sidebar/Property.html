属性面板

属性面板

可视化编辑场景元素的属性。

文本属性

点击输入文本。

展示属性

只能展示，无法编辑。

数字属性

输入数字，点击右侧三角可以改变数字，滚动滚轮也可以改变数字。有的数字控件只允许输入整数。

复选框属性

点击勾选和取消勾选。

颜色属性

点击选择颜色。
