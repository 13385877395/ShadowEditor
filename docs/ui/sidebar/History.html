历史面板

历史面板

查看场景编辑历史，可以快速跳转到某个历史状态。

撤销

点击编辑菜单撤销按钮。撤销的历史以灰色显示。

重做

点击编辑菜单重做按钮。

清空历史

点击编辑菜单清空历史记录按钮或历史面板清空按钮。

跳转历史

点击历史面板上的历史，可以立刻让场景跳转到某个历史状态。
