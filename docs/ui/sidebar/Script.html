脚本面板

脚本面板

可以创建脚本，使用脚本编辑器编辑脚本。

创建脚本

点击创建按钮，可以创建脚本。

编辑脚本

点击脚本右侧编辑按钮，可以编辑脚本。

删除脚本

点击脚本右侧删除按钮，可以删除脚本。
