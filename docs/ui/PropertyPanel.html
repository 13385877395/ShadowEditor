属性面板

属性面板

属性面板用于可视化修改场景中各物体的属性，当前系统已经具有33个不同的组件。详见组件一章。

修改控件的值

对于大部分控件，单击控件，可以修改控件的值。控件值的修改，都是立即生效的。

当文本框失去焦点，会更新该属性的值。

对于整数和浮点数控件，既可以单击输入值，也可以按住鼠标左键，水平移动鼠标，快速修改控件的值。
